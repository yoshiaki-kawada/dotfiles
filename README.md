## Installation
```
git clone git@github.com:yoshiaki-kawada/dotfiles.git ~/dotfiles
cd ~/dotfiles
```

### setup all
```
./setup.sh all
```

### setup only dotfiles
```
./setup.sh dotfiles
```

### setup only karabiner
```
./setup.sh karabiner
```

### setup only starship
```
./setup.sh starship
```

### setup only claude
```
./setup.sh claude
```

### setup only vscode
```
./setup.sh vscode
```

### export vscode extensions
```
code --list-extensions > ~/dotfiles/vscode/extensions
```
